Wigner D-matrix.

wignerd
Syntax
D = wignerd(J,angles)
D = wignerd(J,angles,'-')
D = wignerd(J,angles,'+')

d = wignerd(J,beta)
d = wignerd(J,beta,'-')
d = wignerd(J,beta,'+')
Description

wignerd computes the (2J+1)x(2J+1) Wigner rotation matrix DJ(α,β,γ) with matrix elements DJm1,m2(α,β,γ) where m1 and m2 run from J to -J.

J is a positive multiple of 1/2 (0, 1/2, 1, 3/2, 2, etc). angles is an array of values of the three Euler angles, [alpha beta gamma]. Alternatively, only beta can be given, which is equivalent to [0 beta 0].

The third argument, '-' or '+', specifies the sign convention to use. If omitted, '-' is the default. The matrix elements are defined by

The '-' convention uses the minus signs in the exponents and is used in the books by Brink and Satchler, Zare, Sakurai, Varshalovich, and others (see below for references). The '+' convention uses the plus signs in the exponents and is used in the book by Edmonds, among others.

If only one angle is given (beta, equivalent to [0 beta 0]), the Wigner small-d matrix is computed.

In all cases, the resulting (2J+1)x(2J+1) matrix is arranged such that the indices m1 and m2 run from +J to -J from top to bottom and from left to right, respectively.

Examples
wignerd(1/2,[pi/3 pi/3 pi/4])

ans =
   0.5272 - 0.6871i  -0.4957 + 0.0653i
   0.4957 + 0.0653i   0.5272 + 0.6871i

wignerd(1/2,[pi/3 pi/3 pi/4])

 ans =
   0.5272 - 0.6871i  -0.4957 + 0.0653i
   0.4957 + 0.0653i   0.5272 + 0.6871i

wignerd(2,pi/2)

ans =
    0.2500   -0.5000    0.6124   -0.5000    0.2500
    0.5000   -0.5000   -0.0000    0.5000   -0.5000
    0.6124    0.0000   -0.5000   -0.0000    0.6124
    0.5000    0.5000    0.0000   -0.5000   -0.5000
    0.2500    0.5000    0.6124    0.5000    0.2500
Algorithm

wignerd does not use the explicit scalar expressions given in many books on angular momentum. Instead, it numerically computes the matrix exponentials using matrix representations for Jz and Jy and obtains the full D matrix for a given J in one go.

Here are a few references that contain substantial theory about the Wigner D-functions.

  • A.R. Edmonds, Angular Momentum in Quantum Mechanics, Princeton University Press, 1996.
  • M.E. Rose, Elementary Theory of Angular Momentum, Wiley, 1967.
  • D.M. Brink and G.R. Satchler, Angular Momentum, 3rd ed., Oxford University Press, 1994
  • D.A. Varshalovich, A.N. Moskalev, V.K. Khersonskii, Quantum Theory of Angular Momentum, World Scientific, 1988.
  • R.N. Zare, Angular Momentum: Understanding Spatial Aspects in Chemistry and Physics, Wiley, 1988.
  • J.J. Sakurai, J. Napolitano, Modern Quantum Mechanics, 2nd ed., Addison-Wesley, 2011.
