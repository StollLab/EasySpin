levelsplot

Display energy level diagram and resonances.

Syntax
levelsplot(Sys,Ori,B)
levelsplot(Sys,Ori,B,mwFreq)
levelsplot(Sys,Ori,B,mwFreq,Par)
Description

Displays the energy level diagram (Zeeman diagram) for the spin system specified in Sys. Ori specifies the magnetic field direction in the molecular frame. B gives the field range, and mwFreq provides the microwave frequency.

Ori can be either an array of numbers or a character arrat. As numeric array, it specifies the Euler angles [phi theta] or [phi theta chi] of the magnetic field relative to the molecular frame of the spin system. The three principal directions x, y, and z of the molecular frame, as well as a few specific intermediate orientations, can be specified by shorthands instead: 'x' for [0 pi/2], 'y' for [pi/2 pi/2], 'z' for [0 0], 'xy' for [pi/4 pi/2], 'xz' for [0 pi/4], 'yz' for [pi/2 pi/4], and 'xyz' for [pi/4 acos(1/sqrt(3))]. If Ori is omitted, 'z' is assumed by default.

B contains the magnetic field range, in mT. If it is an array with two numbers [Bmin Bmax], they are understood to be the lower and upper limit of the field range. If it is a single number, it is understood to be the upper field limit (Bmax), and the lower one is assumed to be zero. The number of field values between the two limits is set automatically, but can be modified via Par.nPoints. If B is a full vector of field values, they are used directly.

If mwFreq, the spectrometer frequency in GHz is given, resonances for this frequency are plotted as well. Their colour indicates the transition amplitude: Red lines indicated allowed resonances, gray ones indicate forbidden ones. If the lines are terminated with dots, the relative transition intensity is above 1%. Clicking on a resonance line will print its relative transition probability to the command window.

Par is an optional structure to provide additional settings. It can have the following fields:

Units
Energy units for plotting. The possible values are 'GHz' (default), 'cm^-1', or 'eV'.
nPoints
Number of points along the field axis.
ColorThreshold
Coloring threshold. If mwFreq is given, then transitions with relative intensity above this value ("allowed" transitions) will be plotted in red. The others will be plotted in gray.
PlotThreshold
Transitons with relative intensity below this value will not be plotted.
Examples

To see resonances at X band for the system in the example for the function levels, use 

mySystem = struct('S',5/2,'g',[2 2 2],'D',[-250 -1750 2000]);
Ori = [0 0]; FieldRange = [0 500]; Freq = 9.5;
levelsplot(mySystem,Ori,FieldRange,Freq);
Algorithm

Energy levels are computed using levels. Resonances are computed using resfields.
