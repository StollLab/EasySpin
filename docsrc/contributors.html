Contributors

The following people have made major contributions to the codebase of EasySpin.

  • Peter Martin: trajectory-based EPR simulations, stochastic trajectories, interface to molecular dynamics
  • Stephan Pribitzer: transition-selective spin operators, pulse EPR simulations, software project infrastructure
  • Claudia Tait: pulse shaping, rf mixer, transmitter nonlinearity, resonator, spin-polarized systems, esfit
  • Matthew Krzyaniak: data import from SpecMan, spin-polarized systems
  • Ashley Redman: function signatures
  • Joscha Nehrkorn: frequency-domain simulations, general excitation geometries, magnetometry, higher-order operators
  • Jeremy Lehner: general stochastic Liouville operators
  • Stefan Stoll

For a full list of contributors, see the EasySpin repository on GitHub.
