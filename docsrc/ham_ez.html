ham_ez
Syntax

Electron Zeeman interaction Hamiltonian and its components. 

H = ham_ez(Sys, B)
H = ham_ez(Sys, B, idx)
H = ham_ez(Sys, B, idx, 'sparse')
[Zx, Zy, Zz] = ham_ez(Sys)
[Zx, Zy, Zz] = ham_ez(Sys, idx)
[Zx, Zy, Zz] = ham_ez(Sys, idx, 'sparse')
Description

ham_ez calculates the electron Zeeman interaction Hamiltonian for the spin system given in Sys.

Sys is a spin system structure.

If idx is omitted, all electron spins defined in the spin system are included in the operators. If idx is given, only the specified subset of electron spins is included. For a system with n electron spins, a number between 1 and n indicates an electron spin.

If a magnetic field vector (in mT) is given in B, the complete electron Zeeman Hamiltonian is returned

If no magnetic field is given, the function returns the three operators involved in constructing the electron Zeeman interaction Hamiltonian. These three operators Zx, Zy and Zz are the cartesian components of the vector derivative , i.e.

They correspond to the negative of the electron magnetic moment components. To get the complete electron Zeeman interaction Hamiltonian, they have to be multiplied by the respective B field components and summed up.

If 'sparse' is given as third argument, results are returned in sparse format instead of full format.

All quantities are formulated in the molecular frame, i.e. x refers to the x axis of the molecular frame etc. The three component operators are in units of MHz/mT, and the overall operator is in units of MHz.

Examples

For the spin system 

ABC = struct('S',1/2,'g',[2.1 2.1 2],'Nucs','14N','A',[1 1 1]);

the electron Zeeman Hamiltonian for a given magnetic field along the molecular x axis is 

B = [340;0;0];
H = ham_ez(ABC,B)
