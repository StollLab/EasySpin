easyspincompile

Compile EasySpin.

Syntax
easyspincompile
Description

The command tries to compile the components of EasySpin that are platform dependent and need to be compiled: all *.c files in the private subdirectory.

If the mex system in Matlab is not set up, it should be set up first with mex -setup.
