zfsframes

Eigenframes analysis of D tensor

Syntax
zfsframes(D1,D2,D3)
zfsframes([D1,D2,D3])
Description

zfsframes takes the three principal values of a zero-field splitting (D) tensor as inputs and examines all six possible eigenframe alignments, computes the scalar parameters D and E for each of them and determines the conventional ones.

The three input values can be input either as three separate numbers or as an array of three numbers.

zfsframes does not return any output data, but it prints all its analysis to the screen.

The two axis conventions are

  • Blumberg 1967: requires |Dz|>|Dy|>|Dx| and yields E/D between 0 and 1/3. Most common convention.
  • Poole 1974: requires |Dz|>|Dx|>|Dy| and yields E/D between 0 and -1/3. Occasionally used for organic triplet states, never for transition metals.
Examples

A D tensor with the three principal values 1, 2 and 4 (in arbitrary units) is analysed using 

zfsframes(1,2,4)
or 
D = [1 2 4];
zfsframes(D)
The conventional D and E/D values are D = +2.5 and E/D = +0.2.
