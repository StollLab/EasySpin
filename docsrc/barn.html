barn

Conventional unit of nuclear quadrupole moments in SI units.

Syntax
value = barn
Description

barn returns the SI value of 1 barn (a common unit for nuclear electric quadrupole moments), in m2.

It is defined as 1 barn = 10-28 m2 = 10-24 cm2.

Reference
  • M. G. Holloway, C. P. Baker, How the barn was born, Physics Today 25(7), 9 (1972).
