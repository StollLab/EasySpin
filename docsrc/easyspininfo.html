easyspininfo

Display information about EasySpin installation.

Syntax
easyspininfo
Description

The command displays information about the EasySpin installation on your system, including release number, release date and installation directory.

In addition, it checks for potential name conflicts between EasySpin functions and other functions on the MATLAB path.
