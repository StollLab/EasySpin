nnint

Nuclear-nuclear interaction Hamiltonian.

Syntax
F = nnint(SpinSys)
F = nnint(SpinSys,nucSpins)
F = nnint(SpinSys,nucSpins,'sparse')
Description

nnint returns the nuclear-nuclear interaction part of the spin Hamiltonian of the spin system given in SpinSys, in units of MHz.

In the spin system structure, the fields nn and nnFrame parameterize the nuclear-nuclear interaction.

If the vector nucSpins is specified, only the interactions between the nuclei listed in nucSpins is computed. For example, if nucSpins = [1 3], only the interaction between nucleus 1 and nucleus 3 is returned.

If 'sparse' is given as third argument, the Hamiltonian matrix is returned in sparse format, instead of full format.

Examples

For a system with one electron and two nuclei 

Sys.S = 1/2;
Sys.Nucs = '1H,1H';
Sys.nn = [1 1 -2]*0.04;   % MHz
Sys.A = [10 12];          % MHz

the nuclear-nuclear interaction Hamiltonian (in MHz) is 

nnHam = nnint(Sys)
