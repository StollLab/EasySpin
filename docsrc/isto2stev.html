isto2stev

Transformation matrix from irreducible spherical tensor operators to Stevens operators.

Syntax
Ck = isto2stev(k)
Description

This function calculates the transformation matrix Ck to transform an irreducible spherical tensor operator (ISTO) of rank k, Tk, to a vector of rank-k extended Steven operators Ok with Ok = Ck * Tk.

Tk is the ISTO of rank k with 2k+1 components T(k,q=k), T(k,k-1), etc. Ok is the vector of Stevens operators O(k,q=k), O(k,k-1), ..., O(k,0), ..., O(k,-k).

Stevens operators with non-negative q are cosine tesseral operators, and those with negative q are sine tesseral operators.

Ck is the (2k+1)x(2k+1) transformation matrix. Its only non-negative elements are on the diagonal and on the antidiagonal.

To obtain the matrix for the inverse transformation (from Stevens operators to ISTOs), use inv(Ck): Tk = inv(Ck)*Ok.

Examples

The transformation matrix for a rank-2 ISTO is 

C2 = isto2stev(2);
