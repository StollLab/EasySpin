nucabund

Natural abundance of nuclear isotopes.

Syntax
Q = nucabund(Isotopes)
Description

This function returns the natural abundance of the nuclei specified in Isotopes, as a number between 0 (not occurring naturally) and 1 (the only naturally occurring isotope). Isotopes is a character array giving the mass number and the chemical symbol of the element, e.g. '14N' or '63Cu'. If Isotopes is a comma-separated list of isotope labels, the function returns a list of all abundances in vector form.

The data are taken from an internal database through nucdata.

Examples
w = nucabund('63Cu,65Cu')

w =
    0.6917    0.3083

sum(w)

ans =
     1
