isotopes

Interactive table of nuclear isotopes.

Syntax
isotopes
Description

This graphical interface provides an interactive periodic table of the elements with a list of nuclear isotope data, including spin, nuclear g factor, gyromagnetic ratio, electric quadrupole moment and resonance frequency.

EasySpin's nuclear isotope database includes all stable isotopes plus the most important radioactive ones.
