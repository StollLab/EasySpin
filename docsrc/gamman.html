gamman

Nuclear gyromagnetic ratios.

Syntax
value = gamman(Iso)
Description

gamman returns the value of the gyromagnetic ratio for the nuclear isotope, or list of nuclear isotopes, given in Iso in SI units (radians per second per tesla) in value.

Examples

The proton gyromagnetic ratio in rad/s/T is 

g1H = gamman('1H')

g1H =
   2.6752e+08

To convert to MHz/T, use 

g1H/2/pi/1e6

ans =
   42.5775

The ratio of the electron gyromagnetic ratio to the proton gyromagnetic ratio is 

gammae/gamman('1H')

ans =
 -658.2107
