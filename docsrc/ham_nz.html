ham_nz
Syntax

Nuclear Zeeman interaction Hamiltonian and its components. 

H = ham_nz(Sys, B)
H = ham_nz(Sys, B, idx)
H = ham_nz(Sys, B, idx, 'sparse')
[Zx, Zy, Zz] = ham_nz(Sys)
[Zx, Zy, Zz] = ham_nz(Sys, idx)
[Zx, Zy, Zz] = ham_nz(Sys, idx, 'sparse')
Description

ham_nz calculates the nuclear Zeeman interaction Hamiltonian for the spin system given in Sys.

Sys is a spin system structure.

If idx is omitted, all nuclear spins defined in the spin system are included in the operators. If idx is given, only the specified subset of nuclear spins is included. idx> must be an array of numbers between 1 and n for a spin system with n nuclear spins.

If a magnetic field vector (in mT) is given in B, the complete nuclear Zeeman Hamiltonian is returned

If no magnetic field is given, the function returns the three operators involved in constructing the nuclear Zeeman interaction Hamiltonian. These three operators Zx, Zy and Zz are the cartesian components of the vector derivative , i.e.

They correspond to the negative of the nuclear magnetic moment components. To get the complete nuclear Zeeman interaction Hamiltonian, they have to be multiplied by the respective B field components and summed up.

If 'sparse' is given as third argument, results are returned in sparse format instead of full format.

All quantities are formulated in the molecular frame, i.e. x refers to the x axis of the molecular frame etc. The three component operators are in units of MHz/mT, and the overall operator is in units of MHz.

Examples

For the spin system 

ABC = struct('S',1/2,'g',[2.1 2.1 2],'Nucs','14N','A',[1 1 1]);

the nuclear Zeeman Hamiltonian for a given magnetic field along the molecular x axis is 

B = [340;0;0];
H = ham_nz(ABC,B)
