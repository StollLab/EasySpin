unitconvert

Convert values between various energy and field units.

Syntax
vals_out = unitconvert(vals, fromto)
vals_out = unitconvert(vals, fromto, g)
Description

unitconvert takes vals and converts its value(s) to a different unit based on the unit conversion given in the string/character array fromto. It returns the conversion result in vals_out.

vals can be a scalar or an array of values.

fromto is a string or character array that specifes the source and the target unit, connected by '->', for example 'MHz->mT' for the conversion from MHz to mT, or 'cm^-1->GHz' for converting from cm-1 to GHz.

The following units are supported: 'cm^-1' (inverse centimeters), 'eV' (electron-volt), 'J' (joule), 'K' (kelvin), 'MHz' (megahertz), 'GHz' (gigahertz), 'THz' (terahertz), 'G' (gauss), 'mT' (millitesla), 'T' (tesla). Conversions are implemented between any pair of these units.

In unit conversions from energy to magnetic field units, or vice versa, the third input g gives the g factor for the conversion. If it is omitted, the g factor of the free electron 2.0023... is used (see gfree). If vals is an array, g can be either a scalar or an array of the same size as vals.

Examples

To convert a zero-field splitting constant of 10 cm-1 to GHz, use 

D = unitconvert(10,'cm^-1->GHz')

D =
  299.7925

To calculate the resonance field of an spin-1/2 system with a g factor of 1.93 at 9.66 GHz, use 

B = unitconvert(9.66,'GHz->mT',1.93)

B =
  357.6089
