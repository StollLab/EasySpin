faraday

Faraday constant.

Syntax
value = faraday
[value,sigma] = faraday
Description

faraday returns the value of the Faraday constant in SI units (Coulomb/mol) in value and the standard deviation in sigma (if two output arguments are specified).

Reference

The value and the standard deviation are taken from the 2014 CODATA collection of recommended values.
